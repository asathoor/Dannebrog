Viser Danmarks flag Dannebrog på en webside.

== Description ==
Denne plugin kan vise Dannebrog på officielle flagdage og fx på din fødselsdag. Det sidste kræver at du tilretter slutningen af filen flag.php.
Programmet kan bruges som:

* Påmindelse om at hejse flaget.
* Din webside flager automatisk på de officielle flagdage.

Denne plugin er gratis. Ønsker du at donere, så støt Danmarkssamfundet, der arbejder for at uddele faner til foreninger, spejdere, sportsklubber mv. i Danmark.

== Installation ==
* Download dannebrog.zip
* Pak mappen ud.
* Upload til ../wp-content/plugins
* Aktiver plugin
* Gå til widgets og placér den hvor du synes.
